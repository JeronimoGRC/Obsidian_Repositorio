## ¿Qué es?

**TTP (HyperText Transfer Protocol)** es el **protocolo de comunicación** entre:

- **Cliente** → tu navegador
- **Servidor** → la web

Es **texto plano**, no cifra nada  y funciona por **peticiones y respuestas**.

----

## Métodos HTTP

- `GET` -> Obtención de datos
- `POST` -> Enviar datos
- `PUT` -> Editar datos
- `DELETE` -> Borrar

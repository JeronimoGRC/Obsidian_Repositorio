
---

## ¿Qué es?

Conjunto de diferentes tipos de datos organizados e interrelacionados.

---

## Principales objetos de una Base de datos


- Tablas.
	- Campos (Columnas) -> Define el tipo de dato.
	- Registros (Filas).
- Relaciones
	- 1 - 1.
	- 1 - varios.
	- varios - varios.
- Índices, claves y restricciones.
- Consultas / Vistas.
- Formularios.
- Otros:
	- Procedimientos / Triggers (disparadores).
